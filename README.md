# Career Roadmap and Internship Tracker

## 📌 Project Overview
Career Roadmap and Internship Tracker is a simple web application that helps students organize their learning journey and keep track of internship opportunities.

## 🚀 Features
- Career roadmap planning
- Internship tracking
- Simple and responsive user interface
- Easy navigation
- Built using HTML, CSS, and JavaScript

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

## 📂 Project Structure
- index.html
- style.css
- script.js

## 🎯 Purpose
This project was created to help students stay organized while preparing for internships and planning their career path.

## 🔮 Future Improvements
- User login system
- Save progress using Local Storage
- Internship application status tracker
- Dark mode
- Search and filter options
